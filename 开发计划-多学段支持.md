# 经费年报生成工具 — 多学段支持开发计划

## 背景

部分学校是多学段学校（九年制 / 十二年制 / 完全中学等），需要在政府系统填报多套报表：
- 综表（自动汇总，"_只读"）
- 各学段分表（用户实际需要填）

源 Excel 财务数据是学校总账（不分学段），需要按规则分摊到各学段。

---

## 数据来源

| 数据 | 来源 |
|---|---|
| 年初学生数 / 教职工数（分学段） | **上年经费年报** |
| 年末学生数 / 教职工数 | **教育事业年报**（学生分学段，教职工只有总数）|
| 财务总账（资产、收入、支出） | 源 Excel 五件套 |
| 生均拨款标准 | **用户手动输入**（参考值，后期可调）|
| 学校学段类型 | **从上年经费年报识别**（也可参考教育事业年报 `bxlx`）|

### 教育事业年报关键字段
- C `bxlx`：办学类型代码
- M：小学学生数；N：初中学生数；O：高中学生数；L：幼儿园学生数
- V/W/X：小/初/高 住宿生
- S/T/U：小/初/高 随班就读
- Y：教职工总数（**不分学段**）
- AA：专任教师总数（**不分学段**）

## 分摊规则

### 学段比例（已确认）

**学生比例**：
```
ratio_学生_X = 该学段学生数 / 总学生数
```
数据从教育事业年报 M/N/O 列取。

**教职工比例（方案 D — 等比缩放，且不超总数）**：
```
1. 从上年经费年报取各学段年初教职工数：T_prev_小, T_prev_初, T_prev_高
2. 计算上年比例：r_X = T_prev_X / sum(T_prev_*)
3. 年末分学段教职工 = 年末总教职工 × r_X
4. 加约束：sum(T_X) <= 年末总教职工，否则按比例缩放
```

### 各表分摊规则

| 表 | 规则 |
|---|---|
| 人员表 | 只综表，填年初+年末（**不拆**）|
| 收入表 | **以支定收**（详细映射规则后续讨论）|
| 支出表（一）工资福利 | 按**教职工比例**分摊 |
| 支出表（二）对个人和家庭的补助 | 按**教职工比例**分摊 |
| 支出表（三）商品和服务 | 按**学生数 × 生均标准**分摊 |
| 支出表（四）资本性支出 | 按**学生数 × 生均标准**分摊 |
| 费用表 | 同支出表规则（待确认）|
| 债务表 | 只综表 |
| 价值量表 | 只综表 |
| 实物量表 | 只综表 |

### 分摊公式

**按教职工比例**：
```
X段金额 = 总额 × r_教职工_X
```

**按学生数 × 生均标准**：
```
权重_X = S_X × R_X    （S=学生数, R=生均标准）
X段金额 = 总额 × 权重_X / sum(权重_*)
```

### "以支定收" 映射（待细化）
初稿假设：
- 一般公共预算教育经费 ← 工资福利 + 对个人补助
- 公用经费 / 生均拨款 ← 商品服务 + 资本性支出
- 其他 ← 0 或源数据原值
